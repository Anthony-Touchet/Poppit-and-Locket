How to Run The Program:
	The program will start off with a small demenstration of moving an object across the screen. To access using simply hit g on the keyborad and a prompt will appear
on the command tab. This is the point where you can use the math the program allows you. While in this phase, THe prompt on the screen will show you what can be done. These 
functions have two different types of outcomes: Vectors/Functions or just a single number. Once A function is preformed, click on the window that has the graph on it. To 
continue using the graph, simply hit g again. To exit, hit Escape.
